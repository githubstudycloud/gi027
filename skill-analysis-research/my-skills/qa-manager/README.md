# Q&A Knowledge Manager for Claude Code

一个强大的问答知识管理系统，帮助你在使用 Claude Code 时有效地保存、组织、搜索和管理问答内容。

## 🌟 核心理念

> "好的问题和好的答案是相辅相成的"

当你在使用 Claude Code 时问了很多问题，这个工具帮你：
- ✅ 保存有价值的问答对
- ✅ 随时搜索和回顾
- ✅ 不断修改和完善答案
- ✅ 构建个人知识库

## 🚀 快速开始

### 1. 立即使用（无需配置）

Skill 已经安装！直接在 Claude Code 中说：

```
保存这个问答：
问题：如何使用 Git rebase？
答案：git rebase 用于重写提交历史...
```

或者搜索之前保存的内容：

```
搜索关于 Docker 的问答
```

就这么简单！

📖 **[快速入门指南](./QUICKSTART.md)** - 5分钟学会所有功能

## ✨ 主要功能

### 1. 💾 保存问答
```
保存这个问答：[问题和答案]
```
- 自动提取标签
- 结构化存储
- 支持代码示例
- 记录时间戳

### 2. 🔍 智能搜索
```
搜索关于 React 性能优化的问答
```
- 语义搜索
- 标签过滤
- 时间筛选
- 模糊匹配

### 3. ✏️ 持续更新
```
更新 React 性能问答，添加新的优化技巧
```
- 版本控制
- 保留历史
- 增量更新
- 状态标记

### 4. 📊 知识管理
```
显示我的问答统计
```
- 分类统计
- 标签分析
- 知识图谱
- 导出备份

## 🎯 适用场景

- **学习新技术**: 保存学习笔记和关键概念
- **解决问题**: 记录问题和解决方案
- **代码片段**: 收集有用的代码示例
- **最佳实践**: 积累项目经验
- **面试准备**: 整理技术面试题
- **团队协作**: 分享知识库

## 🔧 技术实现

### 基于 Claude Code 原生功能

- **Skills**: 自动激活，无需手动调用
- **MCP Memory**: 持久化存储和语义搜索
- **Hooks** (可选): 自动化和增强用户体验

### 数据存储

使用你已配置的 **Memory MCP Server**：
- 问答作为实体存储
- 支持观察记录（observations）
- 自动关联和标签
- 持久化到本地数据库

## 💡 最佳实践

1. **及时保存**: 遇到有用答案立即保存
2. **使用标签**: 为每个问答添加相关标签
3. **定期复习**: 每周查看新保存的问答
4. **持续更新**: 发现更好的解决方案时更新
5. **建立关联**: 链接相关问答构建知识网络

## ⚙️ 高级配置（可选）

想要更自动化的体验？配置 Hooks：

1. 编辑 `.claude/settings.local.json`
2. 添加 hooks 配置（见 [hooks/README.md](../hooks/qa-manager/README.md)）
3. 享受自动化功能：
   - 会话启动时显示统计
   - 智能推荐相关问答
   - 对话结束时提示保存
