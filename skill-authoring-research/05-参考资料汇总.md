## D. 关键概念速查表（自整理）

| 概念 | 一句话说明 |
|------|-----------|
| 渐进加载 | name+description → SKILL.md → references，按需展开 |
| Discovery | Agent 通过 description 决定是否加载 Skill |
| 静默失败 | YAML 错误时不报错，Skill 直接消失 |
| 触发关键词 | description 中"当用户说……时"列出的词，决定是否被发现 |
| 一层引用规则 | SKILL.md 中只能引用直接子目录文件 |
| 用户级 vs 项目级 | `~/.claude/skills/` 跨项目；`.github/skills/` 仅本项目 |

## F. 推荐学习顺序

1. 读本研究的 [01-什么是Skill.md](./01-什么是Skill.md) → 建立心智模型
2. 读 `~/.claude/skills/adr/SKILL.md` → 看真实工作流型 Skill
3. 读 `agent-customization/references/skills.md`（VS Code 扩展自带）→ 对照规范
4. 复制本研究的 [04-模板库.md](./04-模板库.md) 中适配模板，改成自己的
5. 用 [03-编写技巧与最佳实践.md#36-自检清单](./03-编写技巧与最佳实践.md) 自检
6. 在 VS Code 输入 `/` 验证 → 用真实触发词测试
