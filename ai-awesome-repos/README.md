## 大语言模型 LLM

### 1. [ollama](https://github.com/ollama/ollama) ⭐ 169,663
- **语言**: Go
- **简介**: 在本地快速运行 Kimi-K2.5、GLM-5、MiniMax、DeepSeek、gpt-oss、Qwen、Gemma 等大模型
- **特点**: 一键部署本地大模型，无需复杂配置

### 2. [transformers](https://github.com/huggingface/transformers) ⭐ 159,714
- **语言**: Python
- **简介**: 最先进的机器学习模型框架，支持文本、视觉、音频和多模态模型
- **特点**: Hugging Face出品，业界标准的模型定义框架

### 3. [DeepSeek-V3](https://github.com/deepseek-ai/DeepSeek-V3) ⭐ 102,712
- **语言**: Python
- **简介**: DeepSeek发布的强大开源大语言模型
- **特点**: 性能媲美闭源模型，完全开源

### 4. [llama.cpp](https://github.com/ggerganov/llama.cpp) ⭐ 105,612
- **语言**: C++
- **简介**: C/C++实现的LLM推理引擎
- **特点**: 极致性能优化，支持各种硬件加速

### 5. [vllm](https://github.com/vllm-project/vllm) ⭐ 77,627
- **语言**: Python
- **简介**: 高吞吐量、内存高效的LLM推理和服务引擎
- **特点**: 生产级LLM部署方案

## 趋势分析

### 2026年AI开源趋势

1. **AI Agent爆发**: Agent相关工具占比超60%，从模型训练转向应用落地
2. **多模态融合**: 文本、视觉、音频模型统一框架成为主流
3. **本地化部署**: Ollama、LocalAI等本地运行方案受到广泛关注
4. **RAG普及**: 检索增强生成成为企业AI应用的标准配置
5. **低代码平台**: Langflow、Dify等可视化开发平台降低AI应用门槛
